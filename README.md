# Personalized-F1-Dashboard-
Personal F1 2026 dashboard — live countdown, standings, calendar &amp; paddock intel. Red Bull themed. Built for Meghasai.

A personal Formula 1 2026 season dashboard built as a single self-contained HTML file. 
Features a live race countdown, full season calendar, drivers' and constructors' championship 
standings, podium results, and paddock news — all themed around Red Bull Racing and Max Verstappen. 
No frameworks, no dependencies, no build step. Just open in a browser.
